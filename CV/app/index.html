Work boards

Check on Github Open Demo

Hi! I want to present a Work Boards. It is just a simple, but quite usefull project to demonstrate my skills. Work Boards is a SPA (Singla Page Application) based on MVC framework - Backbone.js with a Mationette extention, and underscore templating system. App stores all data on real-time database - Firebase, and uses a single profile, so everyone have an acces without login to data (for now, it is just a simplification). Useful techniques, which I used in this projest is a SASS (CSS preprocessor) and Gulp to compile it, minimalize the code and make a distribution build. Work boards project is design with full RWD support and can be used on mobile devices with its full usability.
